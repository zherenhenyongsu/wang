    一.算数符分类
        +  进行加法运算
            1.当符号两边都是数字或者布尔的时候,会进行数学运算
            // true+true=2  false+false=0  false+
            2.只要任何一边是字符串的时候,就会进行字符串拼接
        -  减法运算 ]
        *  乘法     ]
        /  除法     ]
        %  取余     ]!!!
                []四种只能进行数学运算

        

    二.赋值运算符
        =  赋值
        

    三.比较运算符
       
!!!比较运算符得到的结果一定是一个布尔值!!!

        1.>  <  >=  >=  与数学中一致
        
        2.==   不完全等于
        3.===  完全等于
        4.!=   完全不等于
        5.!==  不完全不等于  
        

    四.逻辑运算符
       1.&&  与运算
        
       2.||  或运算
        
       3.!   非运算
        
    
    五.自增自减运算符
    1.++
